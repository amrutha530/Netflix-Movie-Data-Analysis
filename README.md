# 🎬 Netflix Movie Data Analysis

## 📌 Project Overview
This project analyzes a movie dataset using Python to identify trends in movie popularity, ratings, genres, and languages through Exploratory Data Analysis (EDA).

## 🛠️ Technologies Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Google Colab

## 📊 Project Workflow
- Data Collection
- Data Cleaning
- Exploratory Data Analysis (EDA)
- Data Visualization
- Insights & Conclusion

## 📈 Key Insights
- English is the most common original language.
- Most movies have ratings between 6 and 8.
- A few movies have exceptionally high popularity compared to others.
- Visualizations helped identify trends in ratings and popularity.

## 📂 Files
- `Movie_Data_Analysis.ipynb` – Complete notebook with code and analysis.
